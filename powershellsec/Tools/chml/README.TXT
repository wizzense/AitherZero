
CHML.EXE is Mark Minasi's better version of Microsoft's ICACLS.EXE for viewing and editing mandatory integrity control labels in Windows Vista and later (see Mark's book for a full discussion: "Administering Vista Security: the Big Surprises").

REGIL.EXE is Minasi's tool for mandatory integrity control labels on registry keys.

Download the latest versions for free from Mark's web site: http://www.minasi.com/apps/
