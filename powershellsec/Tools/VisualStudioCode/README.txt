Microsoft Visual Studio Code (VSCode) is a free, cross-platform
software development editor.  VSCode supports many languages, 
including PowerShell:

    https://code.visualstudio.com

Make sure to install the PowerShell extension for VSCode:
click the Extensions icon, pull down the [...] menu,
choose "Install from VSIX", and choose the VSIX file
for the PowerShell extension in this folder.
