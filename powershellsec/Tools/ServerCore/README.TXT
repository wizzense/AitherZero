Corefig is a GUI tool to manage Server Core locally.

https://github.com/ejsiron/Corefig


Instructions:

Create a new folder, e.g., C:\Corefig.
Mount the corefig ISO inside the VM.
Copy the contents of the ISO into that folder.
Open CMD.EXE elevated and switch into that folder.
First run: "cscript.exe Start_corefig.wsf"
Subsequent runs: "powershell.exe .\corefig.ps1"
