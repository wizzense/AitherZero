### Reports Not Updating?

1. **Check if workflows are running:**
   ```bash
   gh run list --workflow=publish-test-reports.yml
   ```

2. **Manually trigger report generation:**
   ```bash
   gh workflow run publish-test-reports.yml
   ```

3. **Generate reports locally:**
   ```powershell
   ./library/automation-scripts/0512_Generate-Dashboard.ps1 -Format All
   ```

### Old Reports Still Showing?

Old test reports (TestReport-*.json from specific dates) are ephemeral and should not be committed. They are automatically cleaned up in `.gitignore`.
